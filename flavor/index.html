FFR or IU to Guide PCI

Background

  • CAD 환자에서 PCI를 평가할 때, FFR 또는 IVUS를 이용하여 재혈관화 및 스텐트 삽입 여부를 결정할 수 있음

  • 그러나, 이 두 방법 중 하나만을 사용하여 전체 과정을 진행했을 때 임상 결과에 어떤 차이가 있는지는 명확하지 않음

Methods

  • 중등도 협착(관상동맥조영술에서 시각적으로 40~70% 폐색으로 추정됨)에 대해 PCI 여부를 평가받는 1682명의 환자를 1:1 비율로 무작위로 배정하여 FFR(혈류분획예비력) 또는 IVUS(혈관내 초음파)로 유도된 시술을 받도록 함

  • FFR 또는 IVUS는 PCI 시행 여부 결정과 시술 성공 여부 평가에 사용

Methods

선정 기준

  • 만 19세 이상

  • 허혈성 심장질환이 의심됨

  • 관상동맥조영술에서 다음이 확인됨:

    • 새롭게 발생한 중등도 협착(40~70%)

    • 협착 부위의 혈관 직경이 최소 2.5mm (시각적 추정 기준)

Methods

제외 기준

  • 비심장성 동반 질환으로 기대 여명이 2년 미만

  • 병변이 다음 위치에 있는 경우:

    • 좌주관상동맥

    • 관상동맥 우회 이식 혈관

  • 출혈 위험이 높은 환자

Methods

  • FFR군에서는 FFR 값이 0.80 이하일 경우 PCI를 시행

  • IVUS군에서는 최소 내경 면적이 3mm² 이하이거나, 3~4mm²이면서 플라크 부하(plaque burden)가 70%를 초과할 경우 PCI를 시행

Methods

FFR 군

  • PCI 기준:

    • FFR ≤ 0.80 (혈류 감소 ≥ 20%)

  • PCI 성공 기준:

    • 시술 후 FFR ≥ 0.88

    • 스텐트 양끝 FFR 차이 < 0.05

Methods

IVUS 군

  • PCI 기준:

    • MLA ≤ 3 mm²

    • 또는 3 < MLA ≤ 4 mm², Plaque 단면적 > 70%

  • PCI 성공 기준:

    • 스텐트 양끝 단면적 ≤ 55%

    • 최소 스텐트 면적 ≥ 5.5 mm² 또는 스텐트 끝 혈관 단면

Methods

  • 1차 결과(primary outcome)는 무작위 배정 후 24개월 이내의 사망, 심근경색, 또는 재혈관화로 구성된 복합 지표

  • FFR군이 IVUS군에 비해 열등하지 않음을 검증하기 위해 비열등성을 평가, 비열등성의 허용 한계는 2.5%포인트

Methods

1차 결과

  • 다음 세 가지 사건을 포함한 복합 지표 (24개월 시점 기준):

    • 모든 원인에 의한 사망

    • 심근경색

    • 어떤 형태의 재혈관화

Methods

2차 결과

  • 주요 결과의 개별 구성 요소

  • 스텐트 사용:

    • 환자 1명당 스텐트 수

    • 혈관 1개당 스텐트 수

  • 뇌졸증 발생 여부

  • SAQ

Results

  • PCI 시행 비율은 FFR군에서 44.4%, IVUS군에서 65.3%

  • 24개월 시점에서 1차 결과 사건(사망, 심근경색, 또는 재혈관화 발생)은 FFR군의 8.1%, IVUS군의 8.5%에서 발생(절대 차이 −0.4%포인트; 단측 97.5% 신뢰구간의 상한값 2.2%포인트; 비열등성에 대한 P값 = 0.01)

  • 시애틀 협심증 설문(Seattle Angina Questionnaire)을 통한 환자 자가 보고 결과는 두 군 간 유사

Conclusions

  • 중등도 협착으로 PCI 여부를 평가받는 환자에서, FFR 유도 시술은 사망, 심근경색 또는 재혈관화로 구성된 24개월 시점의 복합 1차 결과에 대해 IVUS 유도 시술에 비해 비열등함
